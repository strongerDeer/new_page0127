## Props 설명

- `label` - 입력 필드 레이블
- `state` - 입력 필드 상태 (default, error, success)
- `disabled` - 비활성화 여부
- `full` - 전체 너비 사용 여부
- `errorMsg` - 에러 메시지
- `helperMsg` - 도움말 메시지
