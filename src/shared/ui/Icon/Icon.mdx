# Icon

아이콘 컴포넌트입니다. [Tabler Icon](https://tabler.io/icons)을 사용합니다.

<Canvas>
  <Story of={IconStories.Default} />
</Canvas>
<Controls />

## 특징

- SVG 기반의 아이콘 시스템
- 크기 조절이 자유로움
- 색상 커스터마이징 가능
- 접근성 고려(추후 개선)

## 사용법

```tsx
import Icon from '@/shared/ui/Icon';

function MyComponent() {
  return (
    <>
      {/* 기본 사용 */}
      <Icon name="heart" />

      {/* 크기 변경 */}
      <Icon name="heart" size={24} />

      {/* 색상 변경 */}
      <Icon name="heart" color="#FF0000" />
    </>
  );
}
```

## 아이콘 종류

- 아이콘을 클릭하면 해당 코드가 복사됩니다.

<div
  style={{
    display: 'grid',
    gridTemplateColumns: 'repeat(auto-fill, minmax(6rem, auto))',
    gap: '1em',
  }}
>
  {Object.keys(ICONS).map((icon, index) => (
    <button
      style={{ background: 'none', border: 0, cursor: 'pointer' }}
      onClick={() => {
        window.navigator.clipboard.writeText(`<Icon name=${icon} />`);
      }}
    >
      <p style={{ fontSize: '1.2rem', margin: '0 0 0.5em' }}>{icon}</p>
      <Icon name={icon} key={index} size={32} />
    </button>

))}

</div>
